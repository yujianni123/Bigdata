# 校园反诈智能预警平台项目介绍

## 项目概述

校园反诈智能预警平台是一个专注于预防和识别校园诈骗的智能系统，通过整合多种数据源和分析技术，为高校师生提供诈骗风险评估、案例学习和预警管理等功能。

## 技术架构

### 后端技术栈

- **框架**: Flask 2.3.3
- **认证**: Flask-JWT-Extended 4.4.4
- **数据库**: SQLite (本地存储)
- **限流**: Flask-Limiter 4.1.1
- **CORS**: Flask-CORS 3.0.10
- **监控**: Prometheus Client 0.23.1
- **环境管理**: python-dotenv 0.19.2

### 前端技术栈

- **HTML5/CSS3**: 基础页面结构和样式
- **Bootstrap 5.3.0**: 响应式UI框架
- **JavaScript**: 前端交互逻辑
- **Chart.js**: 数据可视化
- **Font Awesome**: 图标库

## 核心功能

### 1. 用户认证与授权

- **用户注册**: 支持新用户注册
- **用户登录**: JWT令牌认证
- **角色管理**: 支持管理员、教师、学生三种角色

### 2. 诈骗案例管理

- **案例列表**: 分页展示诈骗案例
- **案例详情**: 查看案例详细信息
- **随机案例**: 每日一案例推荐
- **案例搜索**: 支持按关键词搜索案例

### 3. 风险评估

- **文本检测**: 分析文本内容的诈骗风险
- **URL检测**: 分析URL的安全风险
- **风险评分**: 基于多维度指标的风险评分系统
- **相似案例推荐**: 根据检测结果推荐相似案例

### 4. 预警管理

- **个人预警**: 查看个人的风险预警
- **班级预警**: 教师查看班级学生的预警
- **预警处理**: 教师处理学生预警
- **重点关注**: 教师标记重点关注学生

### 5. 数据统计与分析

- **仪表盘**: 总览数据统计
- **诈骗类型分布**: 可视化展示诈骗类型分布
- **诈骗金额统计**: 分析诈骗金额趋势
- **高风险用户分析**: 识别高风险用户

## API接口

### 认证接口

- **POST /api/auth/register**: 用户注册
- **POST /api/auth/login**: 用户登录

### 案例接口

- **GET /fraud-cases**: 获取所有诈骗案例
- **GET /api/cases**: 案例列表（分页）
- **GET /api/cases/<case_id>**: 案例详情
- **GET /api/cases/random**: 随机案例

### 检测接口

- **POST /api/check/text**: 文本检测
- **POST /api/check/url**: URL检测

### 预警接口

- **GET /api/warnings/my**: 个人预警
- **POST /api/warnings/<warning_id>/handle**: 处理预警

### 管理接口

- **GET /api/admin/overview**: 总览数据
- **GET /api/teacher/dashboard**: 教师仪表盘
- **GET /api/teacher/students**: 学生列表

## 数据流程

1. **数据采集**: 通过爬虫从微博等平台采集诈骗相关数据
2. **数据转换**: 将采集的数据转换为标准格式
3. **数据存储**: 存储到本地CSV文件和SQLite数据库
4. **数据分析**: 对数据进行分析和风险评估
5. **数据展示**: 通过前端界面展示分析结果

## 安全措施

- **JWT认证**: 保护API接口安全
- **API限流**: 防止恶意请求
- **输入验证**: 验证用户输入数据
- **密码加密**: 使用pbkdf2:sha256加密存储密码
- **CORS配置**: 安全的跨域请求处理
